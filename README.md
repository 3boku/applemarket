# 사용법

1. 회원가입란에 들어가서 회원가입을 한다.
2. 회원가입을 완료하면 로그인하기에 들어가서 로그인을 한다.
3. 로그인을 마치고(원래 아무런 애니메이션이 없음) 게시물올리기로 이동한다.
4. 게시물을 올리고 '사과마켓'을 눌러 메인페이지로 이동하고 마음것 물고 뜯고 맛보고 즐긴다.
